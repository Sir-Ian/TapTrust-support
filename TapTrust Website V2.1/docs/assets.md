# TapTrust Website Assets

The site uses clean public aliases for the current app icon and homepage
screenshot so source files do not need URL-encoded filenames with spaces.

- `/taptrust-icon-v3.png` aliases
  `/TapTrust Icon V3-iOS-Default-1024x1024@1x.png`.
- `/taptrust-home-screen-iphone-17-pro.png` aliases
  `/Simulator Screenshot - iPhone 17 Pro - 2026-06-08 at 20.13.28.png`.

Keep the original exported files available until the human reviewer confirms
the aliases are preferred for production.
